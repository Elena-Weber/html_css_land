Quick overview:

Ruby is a high-level general-purpose programming language. It was developed by Yukihiro Matsumoto in 1995. Ruby has an elegant syntax that is natural to read and easy to write. It's an open source language meaning it is free to copy, use, modify and distribute. Ruby is considered by a number of programmers and developers today and comes with a wide range of features. It is consistent in nature and has a strong developer community. Start learning Ruby with us!
